 <p>
   <b>Краткое описание:</b><br>
   Элементарное воспроизведение .mp3 и .ogg файлов через <b>pygame</b>. <br><br>

 <h3>Функционал</h3>
 
 | Клавиши     | Описание            |
 | ----------- | --------------------|
 | `SPACE`     | Следующая песня     |
 | `ARROW_UP`  | Увеличить громкость |
 | `ARROW_DOWN`| Уменьшить громкость |
